Plugin Installation

Unzip the downloaded package. Find the zipped theme file called wpkit-elementor.zip.

Go to “Plugins” page in your dashboard side menu. Click “Install Plugin” button and upload the zipped plugin  file. Activate it after uploading is finished.

You can also unzip the installation zip file and upload the plugin folder to “/wp-content/plugins” directory via FTP client.

Modules

In the current version, we include 2 modules: Flexible Panel For Elementor, Global Blocks Module. You can enable/disable them at any time via WPKit For Elementor Page.

Global Blocks

You can create the global blocks by Elementor Page Builder and use these blocks anywhere for the global template parts.

After enabled this module, you will see the global blocks menu item will be shown in the left sidebar, and we recommend you to re-save the permalink via Settings > Permalink in order to make the Elementor works properly for the global block post.

Flexible Panel For Elementor

The Elementor panel is fixed to the left side of the page by default, if you enable this module, the panel will be dragable such as using the Photoshop, and the panel skin will be changed to dark style.

Elementor Widgets

We includes 23 custom Elementor widgets and 2 custom controls in this plugin. After you activated this plugin, the custom widgets will be automatically appended to the panel that you can directly use them to build the page.

Please note, only when you activate WooCommerce, Easy Digital Downloads and Contact Form 7, the related widgets will be displayed and available to use.

In Advanced tab of each widget, you will see Parallax and Opacity sections are added, so you can change the opacity or set the parallax animation for all widgets.

Customize The Widget Template

In some cases, you may need to customize the widget output template in order to match your theme style or layout. Please following the method below:

– Create “wpkit-elementor” folder in your theme folder.

– Copy the templates which you want to customize from the plugin folder: “wp-content/plugins/wpkit-elementor/templates” to your “theme folder/wpkit-elementor”.

– Just edit the copied version, it will override the original template which are included in the plugin folder.

Localization

This theme is ready for localization. We have made every string translate able with the text domain “wpkit-elementor”, use Poedit  editor to translate it, or install Loco Translation plugin to translate the text.

If you choose use Poedit to create your own language file, please do it as following steps

  1. Go to /wp-content/plugins/wpkit-elementor/languages directory, you will find the wpkit-elementor.pot file
  2. Just base on this file template and create your language file. (ie. pt_BR.po, de_DE.po, etc.)
  3. Then, open this file with Poedit editor, translate all words to your language, save the file.
  4. Go to “Setting > General > Site language”, you will find the language options at the bottom, select your language.

Upgrade

If you have registered your purchased code, you can enjoy the plugin auto update. When we release the new version, you only need to click upgrade the plugin via Plugins page.

For manually upgrade the plugin, you can get the new version on your CodeCanyon download page. Please note that if you haven’t modified any source files of the plugin, just remove the old version and reinstall the new version directly, if you have changed some source files, you can’t simply reinstall the new version of the plugin, it’s hard to upgrade smoothly, you must move all your changed codes to the new version of the plugin before you reinstall or override the old files.
